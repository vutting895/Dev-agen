🚀 Quick Start with Docker

Run Dev-Agen in seconds:

docker pull ghcr.io/vutting895/dev-agen:latest docker run -d -p 80:80 ghcr.io/vutting895/dev-agen:latest

Or clone and build locally:

git clone https://github.com/vutting895/Dev-agen.git cd Dev-agen docker build -t dev-agen . && docker run -p 80:80 dev-agen
